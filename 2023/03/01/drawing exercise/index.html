PS常用快捷键

  • V：移动
  • M：选框
  • L：套索
  • I：吸管
  • B：画笔
  • E：橡皮擦
  • G：油漆桶
  • P：钢笔
  • Z：缩放
  • Ctrl + G：新建图层组
  • Ctrl + E：合并选中图层或组
  • Ctrl + J：复制选中图层
  • Ctrl+T：自由变换
  • Ctrl+A：全选
  • Ctrl+D：取消选择
  • 移动画布：Spacebar+鼠标拖动
  • Ctrl + Alt + Shift + N：新建空白图层
  • Ctrl + H：显示/隐藏辅助线
